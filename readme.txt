How to run ??
-------------
PreRequisites :
nand2tetris folder (with existing tools and projects)

Part A_Computer :
Load computer.hdl chip through "load chip" option in Hardware simulator tool inside the nand2tetris tools folder. Upload the user's test script (tst file). Change run time to maximum. Run it. And change the output view from screen to output.

Part B_PongGame :
Load the folder containg all the VM files (Ball, Player, Main and Game) through the "Load folder" option in VM emulator tool inside nan2tetris tools folder, and run it.
